# 项目简介

爱酱手绘图标，这是一个我特别喜欢的图标包，然而现在不更新小米版本了，于是有了这个项目。

本项目提取过往版本的图标文件，并打包为magisk模块，方便使用。

模块根据另一个优秀的项目：完美图标计划，二改而来，与原版模块冲突。

本人技术力有限，仅在红米K50U，magisk alpha最新版，HyperOS1下进行测试，不保证其他环境的兼容。
